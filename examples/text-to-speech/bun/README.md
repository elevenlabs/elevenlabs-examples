# ElevenLabs - Bun

1. Clone this repo
2. `cd elevenlabs-examples/examples/text-to-speech/bun`
3. Setup the environment variables
   - `cp .env.example .env`
   - Fill in the values
4. Install the dependencies

   ```bash
   bun install # install the dependencies
   ```

5. Run the script

   ```bash
   bun dev
   ```
